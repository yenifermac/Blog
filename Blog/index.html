Habilidades del Frontend (HTML Y CSS)

HTML (HyperText Markup Language) y CSS (Cascading Style Sheets) son dos tecnologías fundamentales para la construcción de páginas web. Aquí te explico un poco más sobre cada una y otras herramientas que necesitas para comenzar en el mundo de la programacion

GIT

Git es un sistema de control de versiones que protege nuestro trabajo final, así como todos los cambios realizados durante el proceso de desarrollo. Este sistema nos permite gestionar y controlar cualquier fallo que pueda surgir. En esencia, Git nos permite trabajar en un directorio creado específicamente para el proyecto, realizar cambios, guardarlos y compartirlos con otros desarrolladores. De esta manera, Git no solo asegura la integridad de nuestro trabajo, sino que también facilita la colaboración y el intercambio de ideas entre los desarrolladores.


Flujo de trabajo en git


GIT HUB

GitHub es una plataforma en línea que utiliza Git. Permite a los desarrolladores publicar sus proyectos, colaborar con otros, rastrear y solucionar problemas, y gestionar cambios en el código. Funciona como una especie de red social para desarrolladores, facilitando la colaboración y el intercambio de ideas.

HTML

HTML (HyperText Markup Language) es el lenguaje de marcado que se utiliza para estructurar y presentar el contenido en las páginas web. Con HTML, puedes definir el contenido como texto, imágenes, listas, tablas, etc., y cómo se deben organizar y mostrar estos elementos en la página web.



Anatomía de una página web

CSS

CSS (Cascading Style Sheets) es un lenguaje de estilo que se utiliza para dar estilo y formato a las páginas web. Permite cambiar colores, tamaños de texto, fondos, márgenes, y mucho más.


Dentro de CSS, existe una técnica llamada CSS Grid. Esta técnica permite a los desarrolladores crear un diseño flexible y bidimensional para la colocación de los objetos en una página web. En otras palabras, con CSS Grid puedes definir una rejilla con filas y columnas, y colocar elementos en esa rejilla de manera precisa y flexible. Así, CSS con su característica CSS Grid, te da un control completo sobre el diseño de tu página web.


CSS tambien cuenta con el modulo CSS Flexbox, o Caja Flexible, que proporciona una forma más eficiente de diseñar, alinear y distribuir espacio entre los elementos en un contenedor, incluso cuando sus tamaños son desconocidos o dinámicos.

Dimensiones: Flexbox es unidimensional, mientras que CSS Grid es bidimensional. Flexbox establece elementos a lo largo del eje horizontal o vertical, lo que te exige decidir si deseas un diseño basado en filas o en columnas. Por otro lado, CSS Grid te permite trabajar a lo largo de dos ejes: horizontal y verticalmente.


Control de alineación y distribución de espacio: Flexbox ofrece un mayor control sobre la alineación y la distribución del espacio entre los elementos.
